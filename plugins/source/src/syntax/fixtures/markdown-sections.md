# Project Guide

Welcome to the project.

## Installation

Run `cargo install` to get started.

```bash
cargo install nyne
```

## Configuration

Edit `~/.config/nyne/config.toml`:

```toml
[mount]
root = "~/projects"
```

### Advanced Settings

For power users, additional options are available.

## Usage

Start the daemon with `nyne mount`.

## Troubleshooting

Check logs with `RUST_LOG=debug`.
